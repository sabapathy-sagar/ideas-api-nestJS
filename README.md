## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript repository.

A reddit/twitter style app to keep track of posted App Ideas.

### Features

- Authenticate users
- Users can CRUD ideas
- Users can bookmark ideas
- Support for REST
- Support for GraphQL

### Stack

- Database - PostgreSQL
- REST API - NestJS
- GraphQL API - NestJS
